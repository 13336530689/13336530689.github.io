搭建github——hexo博客教程

Oct 4, 2018

1.安装git

https://git-scm.com/download/win

2.install node.js

####(if cmd can not find “node version”,add 环境变量)

https://nodwjs.org/zh-cn/download/

3.install hexo

npm install hexo-cli -g

查看hexo版本


hexo version

4.初始化blog

hexo init myblog(写blog文件名）

cd myblog

npm install

5.start blog


hexo server

6.搭建github pages

a.注册github账号

b.配置登陆免密码

ssh-keygen -t -rsa -C "your email"

cd ~/.ssh

ls -a

copy *.pub中的所有内容

登陆github——settings——ssh and GPG keys——new ssh key

title可以随意写——key中复制*.pub文件中所有内容——add ssh key

c.创建Github远程仓库

命名格式：用户名.github.io

d.配置_config.yml

e.安装git部署插件

npm install hexo-deployer-git --save

f.推送到服务器

删除旧public文件
hexo clean
生成新的public文件
hexo generate 或 hexo g
开始部署
hexo deploy 或 hexo d

备注：每次修改完文章都要执行,回到source的上一级目录

hexo g

hexo d

三、install themes

1.open https://hexo.io/themes/


2.select one themes and operate accoding to its ways


3.hexo g && hexo d
